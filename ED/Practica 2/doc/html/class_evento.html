Detailed Description

Tipo de dato que representa un evento.

Un Evento contiene:

  • un nombre (string)
  • un día de la semana (1-7)
  • una hora de inicio en formato decimal (0.0 - 24.0)
  • una hora de fin en formato decimal (0.0 - 24.0)

Invariante de representación:

  • 1 <= dia <= 7
  • 0.0 <= hora_inicio < hora_fin <= 24.0

Función de abstracción: El objeto Evento representa el intervalo de tiempo [hora_inicio, hora_fin] del día 'dia', dedicado a la actividad identificada por 'nombre'.
