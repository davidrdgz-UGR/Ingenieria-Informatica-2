Detailed Description

TDA AgendaEventos.

La agenda gestiona un conjunto dinámico de eventos mediante un array dinámico redimensionable.

Invariante de representación:

  • 0 <= num_eventos <= capacidad
  • 'eventos' apunta a un array válido de tamaño 'capacidad'
  • Las primeras 'num_eventos' posiciones contienen eventos válidos

Función de abstracción: La agenda representa el conjunto: { eventos[0], ..., eventos[num_eventos-1] }.
